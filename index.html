Publishing

A writing environment that disappears

Block-based editor with real-time collaboration, version history, and an AI refinement panel. Write in "Void" mode for distraction-free composing — all chrome fades, leaving only your words.

  • Block-based content stored as structured JSON
  • Drag-and-drop contextual paywalls
  • Git-like version control with visual diff
  • Inline toolbar on text selection
  • Slash commands for block insertion

Memberships

Subscriptions without the platform tax

Connect your own Stripe account. Free, premium, and custom tiers with dynamic visibility rules. Members pay you directly — Meridian takes nothing.

  • Stripe integration — connect your own account
  • Free, premium, and custom subscription tiers
  • Automatic Stripe Product/Price creation
  • Customer portal for self-service billing
  • Content visibility based on member tier

Newsletters

Send to your audience, not through a middleman

Built-in newsletter engine with visual automation workflows, segment targeting, and deliverability optimization. BIMI/DMARC/DKIM enforcement out of the box.

  • Visual workflow builder for automations
  • Welcome sequences, win-back campaigns
  • Dedicated IP warming automation
  • Gmail/Yahoo 2024 compliance built-in
  • Send via Resend, Postmark, or any provider

Growth Tools

Native referral and viral infrastructure

Tiered referral programs, embeddable subscribe buttons, social proof badges, and one-click subscribe endpoints. Growth tooling that rivals Beehiiv — self-hosted.

  • Tiered and milestone-based referral rewards
  • Embeddable buttons and web components
  • Live subscriber count badges
  • UTM + cookie attribution tracking
  • AI-powered content resurrection for SEO

Analytics

Privacy-first insights, not surveillance

First-party analytics with churn prediction, LTV forecasting, and content velocity metrics. No Google Analytics dependency. Your data stays on your infrastructure.

  • Real-time dashboard with sparklines
  • 30-day churn prediction ML model
  • Cohort-based LTV forecasting
  • Anonymized peer benchmarking
  • SQL interface for custom reports

Quick start with Docker

$ git clone https://github.com/meridian-blog/meridian.git $ cd meridian $ cp .env.example .env # Edit with your settings $ make dev # Starts PostgreSQL + Redis + App # Open http://localhost:8000 — that's it.
